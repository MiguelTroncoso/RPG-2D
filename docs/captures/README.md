# Capturas H9/H9.1

`h9-android-layout-preview.png` es un render de referencia 1920×1080 de la escena `VerticalSlice` con la presentación H9. Sirve para revisar composición, safe area, cámara y escala de controles; no es una captura de un teléfono.

`android-h9/` está reservado para capturas tomadas directamente desde un teléfono Android. En H9.1 permanece sin capturas porque `adb devices -l` no detectó hardware. No se incluye una imagen simulada como evidencia de hardware.

La captura física deberá excluir notificaciones, correos, números de teléfono, identificadores y cualquier información personal. El detalle del bloqueo y el checklist están en `docs/H9_1_PHYSICAL_VALIDATION.md`.
